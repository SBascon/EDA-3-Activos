# Análisis Exploratorio de Datos (EDA) con el Dataset Assets

Este repositorio contiene un Análisis Exploratorio de Datos (EDA) aplicado al S&P 500, el Oro y Bitcoin, con el objetivo de comparar su comportamiento en términos de rendimiento, volatilidad y relación entre activos.
El estudio busca aportar una visión clara sobre sus diferencias y su posible papel dentro de una estrategia de inversión diversificada mediante técnicas de análisis y visualización de datos.

## Contenido del repositorio

- **`Analisis_Exploratorio_de_Datos_con_Python.ipynb`**: Un Jupyter Notebook con el análisis exploratorio de datos del dataset Assets.
- **`Analisis_Exploratorio_de_Datos_con_Python.py`**: Un script en Python que realiza el mismo análisis que el notebook.
- **`Assets.csv`**: El dataset Assets utilizado para el análisis.
